안녕하세요.

끊임없는 배움을 좋아하는 프론트엔드 개발자 박태원입니다.
현재 저는 네오펙트라는 재활 헬스케어 회사의 B2C팀에서 프론트엔드 포지션의 웹개발을 맡고 있으며 Neofect Portal이라는 사이트의 신규개발과 운영을 vuejs를 사용하여 개발하고 있습니다.
현 회사에서는 atlassian의 jira와 bitbucket을 사용하여 스프린트 단위로 개발을 진행하는 것을 경험중입니다.

회사 외적으로는 react와 apollo를 사용하여 개인 프로젝트를 진행하며 서비스를 운영배포중이며, class component와 functional component를 이용한 효율적인 웹 컴포넌트 설계를 공부중입니다. 최근에는 react의 hook을 이용하여 life cycle이 필요없는 컴포넌트의 최적화를 위해 기존 react 코드를 수정하여 업데이트하였습니다.

java와 php를 다루는 웹 개발자로 시작하여 미대를 재학중에 웹 디자이너로서의 경험을 쌓고 그 이후에 퍼블리셔의 과정을 거쳐 프론트 개발자로 전향했기 때문에 보다 넓은 시야를 갖고 웹개발쪽으로의 다양한 경험치를 갖고 있다는 것이 저의 강점이라고 생각합니다.

제 개인 사이드 프로젝트 위주로만 포트폴리오를 준비하였습니다. 너그러운 마음으로 봐주시면 감사하겠습니다.

[개인] Zomb Movies APP

Zomb Movies APP / DESIGN BY PUMPKIN

Description

개봉 예정의 영화나 최신 영화의 정보, 높은 평점을 받은 영화의 자세한 정보를 얻고 비슷한 내용의 추천영화의 정보까지 얻을 수 있는 기능을 가진 Movis App이다.

2018년 후반부터 많이 들어봤던 graphQL을 연습 해볼겸 시작한 개인 프로젝트였다.
영화를 즐겨보며 주로 블로그나 포털사이트에서 최근 개봉작이나 평점높은 영화의 정보를 얻곤 했었는데, 얼마 전에 TMDB라는 곳에서 배포하는 제작한 영화정보 열람 API를 발견하게 되면서 바로 Zomb Movies APP을 제작하기 시작하였다.

front에서는 react와 react apollo를 사용하였고, backend에서는 graphql과 express를 사용하여 서버작업하였다.

[FullStack] SPA / React / React Apollo / GraphQL / vanilla javascript / html5 / css3

App Features

    • Popular Movies
    • Top Rated
    • Coming Soon
  • Detail Views
    • 영화 상세정보
    • 감독 및 출연진
    • 관련 추천영화
  • Searching
    • 제목으로 영화검색
